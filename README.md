# **🍽️ Restaurant Billing Website**

A complete restaurant billing system built using **HTML, CSS, and JavaScript**.
This application includes **menu management, cart functionality, billing system, and sales reporting**.

---
# 🌐 Live Demo

🚀 Try the application here:
👉 https://billing-website-rust.vercel.app

# **🌟 Features**

* 🍛 **Menu Display** – View items with images, names, and prices
* 🛒 **Shopping Cart** – Add items, adjust quantities, view total
* 💳 **Billing System** – Pay, print bills, and clear cart
* 🛠️ **Menu Management** – Full CRUD (Create, Read, Update, Delete) operations
* 📊 **Sales Reports** – Monthly insights including:

  * Total revenue and order count
  * Item-wise sales breakdown
  * Date-wise sales analysis

# **📖 Usage**

## **🍽️ Viewing Menu**

* Go to **Menu tab**
* Click **Add to Cart** to add items

---

## **🛒 Cart Operations**

* Click **🛒 icon** (bottom-right)
* Use **+ / -** to adjust quantity
* Click:

  * **Pay Now** → Complete purchase
  * **Print Bill** → Generate bill
  * **Clear Cart** → Remove all items

---

## **🛠️ Managing Menu**

* Go to **Manage Menu tab**

**Actions:**

* ➕ Add Item – Enter name, price, image
* ✏️ Edit Item – Modify and update
* ❌ Delete Item – Remove item

---

## **📊 Sales Reports**

* Open **Sales Report tab**
* Select date range
* Click **Generate Report**

**Shows:**

* Total revenue
* Total orders
* Item-wise sales
* Date-wise breakdown
